# Data-Quality ver 0.1 #
A simple tool to apply validation and enrichment from a CSV

## Technical Requirements

* Java 7+ / Clojure 1.8 / Leiningen 2.x

## Usage

* To run the app

    $ java -jar data-quality-standalone.jar data.csv out.json
